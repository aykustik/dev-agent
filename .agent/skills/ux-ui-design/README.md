# UX/UI Design Expertise

## Overview
The UX/UI Design skill provides design principles, usability analysis, color palettes, and style guides for creating user-friendly interfaces.

## Tools
- `get_design_principles` - Get context-specific design principles
- `analyze_usability` - Analyze UI for usability issues
- `generate_color_palette` - Generate cohesive color palettes
- `create_style_guide` - Create basic style guide
- `improve_accessibility` - Accessibility improvement suggestions

## Key Principles
1. **Consistency** - Use familiar patterns
2. **Feedback** - Always inform users of actions
3. **Visibility** - Show system status clearly
4. **Constraints** - Guide users without restricting
5. **Affordance** - Make actions obvious

## Accessibility (WCAG)
- Color contrast minimum 4.5:1
- Keyboard navigable
- Screen reader compatible
- Focus indicators visible
- Text resizable to 200%
